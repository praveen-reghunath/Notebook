
### npm config set a key
```
npm config set registry="https://artifactory.tools.company.com/artifactory/api/npm/company" 
```

### npm config list
```
npm config list
```

### npm config delete a key
```
npm config delete registry
```

#### Change local user name and email

```sh
git config --local user.name "my-user-name"
git config --local user.email "github-email"
```
